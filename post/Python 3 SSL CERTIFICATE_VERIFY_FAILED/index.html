解决Python 3 SSL Certificate问题

发布于 2024-07-17 | 标签: Python 、 Alfred 5 | 1分钟 | 240字数

一句话总结：进入你的Python安装文件夹，找到Install Certificates.command双击安装即可。

最近一直在捣鼓Alfred Workflow，昨天发现了一个workflow是查询Urban Dictionary还蛮有用的，也不用看任何广告了，不过仍需魔法。

第一次使用的时候就发生了问题，打开Alfred的Debug模式发现问题是Python 3的"SSL: CERTIFICATE_VERIFY_FAILED"（忘了截图了），于是上网找了各种方法，要么就是操作过于复杂，要么就是过时无法解决了。

后来在CSDN发现一个最简单的方法，直接进Python的安装文件夹，双击Install Certificates.command即可。

Install Certificates.command

此时再试试Urban Dictionary的workflow已经可以正常使用了。Enjoy it!

不过Power thesaurus的URL依旧拒绝了我，看了原作者的GitHub，有人遇到了跟我一样的问题，说等了24-48小时突然就能使用了，那我只能静静等待了。
